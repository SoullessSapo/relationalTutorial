# relationalTutorial

Pequeño proyecto de ejemplo para explicar **relaciones en bases de datos relacionales** usando **Java 17 + Spring Boot + JPA/Hibernate + PostgreSQL**.  
Incluye entidades con relaciones `@OneToOne`, `@OneToMany`, `@ManyToOne` y `@ManyToMany`, migraciones con Flyway y tests básicos.

---

## 🚀 Objetivos

- Mostrar cómo modelar relaciones entre entidades en JPA.
- Aprender a escribir repositorios y consultas (Spring Data / JPQL).
- Exponer endpoints REST para CRUD y consultas relacionadas.
- Automatizar el esquema con Flyway (opcional).

---

## 🧱 Stack

- Java 17  
- Spring Boot (Web, Data JPA, Validation)  
- PostgreSQL 14+ (o Docker)  
- Flyway (migraciones)  
- Maven Wrapper (`mvnw`)
